🚀 Hackathon Zero – AI Employee System
📌 Overview

Hackathon Zero – AI Employee is a smart AI-powered Employee Management System designed with multiple service tiers: Bronze, Silver, Gold, and Premium.

The platform provides advanced employee solutions with Email and Social Media integration for better communication and scalability.
